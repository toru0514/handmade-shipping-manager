# ユースケース一覧

## 概要

ハンドメイド作品の発送伝票管理システムのユースケースを定義します。
松岡幸一郎氏のドメイン駆動設計入門に基づき、ユースケースを整理しています。

## アクター

| アクター | 説明 |
|---------|------|
| 作家 | ハンドメイド作品を制作・販売する人。発送作業を行う |
| システム | 自動処理を行うシステム（Gmail監視、通知など） |
